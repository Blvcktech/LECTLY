Flow 1: New User — First Visit to First Notes

This is the most important flow. A student discovers Lectly and goes from "what is this?" to "wow, my notes are ready."

SCREEN

Landing Page

Student sees hero, features, pricing. Clicks "Get Started Free"

SCREEN

Sign Up Page

Google/Apple sign-in or email + password. Picks university.

SCREEN

Dashboard (empty)

"No lectures yet. Upload your first one!" Big upload button.

ACTION

Upload Audio File

Drag & drop or tap to select. Pick subject. Optional class code.

STORAGE

Save Audio File

Audio saved to file storage. Lecture record created in database.

API CALL — $

Transcribe Audio (Whisper)

Speech-to-text. ~$0.36 per hour. Takes 1-4 min.

API CALL — $

Generate Notes (LLM)

Transcript → structured notes with headings, key points, definitions. ~$0.10

STORAGE

Save Notes to Database

Notes, transcript, and metadata stored in user's account.

SUCCESS

Notes Ready Screen

"Your notes are ready!" → View Notes button

SCREEN

Notes Viewer

Student reads structured notes. Can use Explain This or enter Learn Mode.

Flow 2: Study Session — Explain This + Learn Mode

Student is reading their notes and wants to understand something better.

SCREEN

Notes Viewer

Student is reading Section 3: "First-Pass Metabolism"

DECISION

Student is confused

Two options: quick explanation or full learning session

Quick Answer
ACTION

Tap "Explain This"

Highlights confusing text. Picks difficulty level.

API — $0.01

LLM Explains

Simpler explanation + real-world analogy returned

RESULT

Explanation Popup

"Think of it like a food delivery..." Student understands. Continues reading.

Deep Learning
ACTION

Tap "Learn Mode"

Enters interactive learning for this section

API — $0.02

LLM Generates Lesson

Step-by-step explanation, analogy, examples, 3 quiz questions

SCREEN

Learn Mode

Student reads lesson, attempts quiz, sees score

RESULT

Quiz Complete

3/3 correct! "Topic Mastered" badge. Back to notes.

Flow 3: Sharing — One Upload, Whole Class Gets Notes

The viral growth loop. One student uploads, shares via WhatsApp, classmates join.

START

Student A's Notes are Ready

Pharmacology lecture processed. Notes look great.

ACTION

Tap "Share Notes"

Gets a share link or class code. Sends via WhatsApp.

OTHER USER

Student B Opens Link

Sees notes preview. Prompted to sign up for full access.

SCREEN

Sign Up (from share)

Quick Google sign-in. Automatically joined to the shared lecture.

RESULT

Student B Gets Notes

Same notes, zero API cost. Can use Explain This and Learn Mode.

repeats for every classmate
GROWTH

200 Students Join from 1 Upload

Cost: ₦320 once. Value: 200 new users. This is your growth engine.

Flow 4: Free → Paid Upgrade

What happens when a free user hits their limit.

SCREEN

Upload Page

Student tries to upload 3rd lecture this month

LIMIT CHECK

Free Limit Reached (2/2)

"You've used all your free lectures this month"

SCREEN

Upgrade Prompt

"Upgrade to Standard for ₦1,500/month and get 10 lectures"

SCREEN

Pricing Page

Compare plans. Select Standard or Pro.

PAYMENT

Paystack Checkout

Card, bank transfer, or USSD. Naira payment.

SUCCESS

Plan Upgraded!

Redirected back to upload page. 10 lectures now available.
